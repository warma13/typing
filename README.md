# 打字练习应用

这是一个功能丰富的中文打字练习应用程序，旨在帮助用户提高中文打字速度和准确性。

**特别说明**：本项目是完全通过与 AI 对话的方式创作完成的，包括代码编写、功能设计和文档整理等各个方面。

## 功能特点

- **多种练习模式**：支持从文章、诗歌、成语等多种类型的内容中进行练习
- **实时反馈**：提供实时的打字速度、准确率统计
- **拼音提示**：在练习过程中显示汉字的拼音，帮助用户学习
- **文章分类**：提供多种类别的练习文章，包括：
  - 古诗词
  - 名人名言
  - 经典小说片段
  - 课本内容
  - 散文
- **数据丰富**：集成了中文新华字典数据，包含大量汉字、词语、成语和歇后语

## 安装与运行

### 方法一：直接运行可执行文件

1. 从 `dist` 目录中找到 `打字练习.exe` 文件
2. 双击运行该文件
3. 应用程序将自动启动并打开练习界面

### 方法二：通过 Python 脚本运行

1. 确保您的系统已安装 Python 3.x
2. 打开命令提示符或 PowerShell
3. 导航到项目目录
4. 运行以下命令：
   ```bash
   python start_app.py
   ```
5. 应用程序将在默认浏览器中打开

### 方法三：通过批处理文件运行

1. 在项目根目录中找到 `dazi.bat` 文件
2. 双击运行该文件
3. 应用程序将在默认浏览器中打开

## 使用指南

1. **选择练习内容**：在左侧面板中选择练习文章类别，然后选择具体文章
2. **开始练习**：在中间的输入框中开始打字
3. **查看统计**：右侧面板将显示实时的打字速度、准确率等统计信息
4. **拼音提示**：当您点击输入框时，上方会显示对应汉字的拼音
5. **完成练习**：完成一篇文章后，系统会显示您的练习成绩

## 数据说明

- **中文新华字典数据**：来源于开源项目，包含了大量汉字、词语、成语和歇后语的详细信息
- **练习文章**：包含了多种类型的中文文本，适合不同水平的用户练习

## 自定义内容

您可以通过以下方式自定义练习内容：

1. **添加新文章**：在 `practice_articles` 目录下的相应子目录中添加新的文本文件
2. **修改现有文章**：编辑 `practice_articles` 目录下的现有文本文件
